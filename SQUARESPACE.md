# D3X Portfolio para Squarespace

Esta carpeta contiene la version frontend-only del portfolio para montar directo en Squarespace.

## Ubicacion

```text
C:\Users\losri\Desktop\CODEX\d3x-orch-main\agency\static\portfolio
```

## Archivos principales

- `index.html`: estructura completa del portfolio.
- `styles.css`: estilos y responsive.
- `script.js`: animaciones, galeria y minijuego.
- `assets/`: imagenes de Delitruck.

## Nota operativa

Esta copia no usa `server.py` ni API local. El contacto queda por email, telefono y LinkedIn para que sea compatible con Squarespace.

## Como trabajarlo desde Codex CLI

```bash
cd C:\Users\losri\Desktop\CODEX\d3x-orch-main\agency\static\portfolio
codex
```

## Montaje recomendado en Squarespace

1. Crear una pagina nueva en Squarespace para `d3x.biz`.
2. Usar secciones nativas para hero, experiencia, proyectos, educacion, credenciales y contacto.
3. Usar bloques de codigo para piezas especiales como galeria, minijuego y animaciones.
4. Subir las imagenes de `assets/` a la biblioteca de Squarespace y reemplazar las rutas locales por las URLs que entregue Squarespace.
