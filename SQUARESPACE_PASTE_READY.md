# Instrucciones Squarespace

Archivos generados para pegar en Squarespace:

- squarespace-body.html -> pegar en un Code Block dentro de la pagina Inicio.
- squarespace-custom.css -> pegar en Design > Custom CSS.
- squarespace-footer.js -> pegar en Settings > Advanced > Code Injection > Footer.

Antes de pegar el HTML final, sube las 3 imagenes de assets a Squarespace y reemplaza estos textos:

- REEMPLAZAR_URL_DELITRUCK_01
- REEMPLAZAR_URL_DELITRUCK_02
- REEMPLAZAR_URL_DELITRUCK_03

Usa la URL publica o interna que Squarespace te entregue para cada imagen.
